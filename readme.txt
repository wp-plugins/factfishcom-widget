== Description ==

= Widget for the Factfish.com API: Display country-infos - free Content for your wordpress =

With an API Factfish.com provides infos about a random selected country incl. the country-flag.
This widget displays this data on a wordpress-website.

= Settings =

The widget-options allow to set

* the widget title

* the language: either english or german

* template: css-settings

* some extratext after the country-infos

* enable / disable caching and set up cachetime

== Installation ==

For detailed installation instructions, please read the [standard installation procedure for WordPress plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

1. Login to your WordPress installation
2. Install plugin by uploading json-content-importer.zip to `/wp-content/plugins/`.
3. Activate the plugin through the _Plugins_ menu.
4. Click on "Design" and then "widget": The "Factfish.com widget" is one of the available widgets.
